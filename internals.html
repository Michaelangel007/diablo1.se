Variables

dlvl

dlvl corresponds to a dungeon level and it holds a value between 0 and 16.

 0 = Tristram
 1 = Cathedral, level 1.
 2 = Cathedral, level 2.
 3 = Cathedral, level 3.
 4 = Cathedral, level 4.
 5 = Catacombs, level 1.
 6 = Catacombs, level 2.
 7 = Catacombs, level 3.
 8 = Catacombs, level 4.
 9 = Caves, level 1.
10 = Caves, level 2.
11 = Caves, level 3.
12 = Caves, level 4.
13 = Hell, level 1.
14 = Hell, level 2.
15 = Hell, level 3.
16 = Hell, level 4.

dtype

dtype corresponds to a dungeon type and it holds a value between 0 and 4.

0 = Tristram  (dlvl:       0)
1 = Cathedral (dlvl:  1 -  4)
2 = Catacombs (dlvl:  5 -  8)
3 = Caves     (dlvl:  9 - 12)
4 = Hell      (dlvl: 13 - 16)

quest_lvl

quest_lvl corresponds to a quest level and it holds a value between 1 and 5.

1 = Skeleton King's Lair
2 = Bone Chamber
3 = Maze
4 = Poisoned Water Supply
5 = Archbishop Lazarus' Lair
