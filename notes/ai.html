Activity Threshold

Each monster has an activity threshold which regulates its range of activities. The activity threshold is sometimes referred to as squelch and it holds a value between 0 and 255.

Inactive

The monster is inactive if its activity threshold is equal to 0.

Active

The monster is partially active if its activity threshold is between 1 and 254, and fully active if its activity threshold is equal to 255.

Initially each monster has an activity threshold of 0.

The activity threshold of each monster is updated during the monster update phase, in one of two ways:

  • If the monster is within line of sight then its activity threshold is set to 255.
  • If the monster is outside of line of sight then its activity threshold is decremented by 1 until it reaches 0.

Behaviors

Most monsters perform actions depending on the distance to their target. For future reference dist will be used to specify this distance, in columns or rows whichever is larger.

The Butcher

The following pseudo-code describes the behavior of The Butcher.

  1. if not standing or inactive; break.
  2. if dist = 1; attack (melee).
  3. else if dist >= 2; walk towards target.
