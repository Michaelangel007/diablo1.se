Chests

Chests always contains their maximum number of items on quest levels.

Small Chest

Small chests contains between 0 and 1 items. On quest levels they always contain 1 item.

Chest

Chests contains between 0 and 2 items. On quest levels they always contain 2 items.

Large Chest

Large contains contain between 0 and 3 items. On quest levels they always contain 3 items.

Chest Item Drops

There is a 12.5% probability that a chest item drop will be a potion or town portal drop and a 87.5% probability that it will be an item drop. On quest levels all chest item drops are magical item drops.

Magical Item Drop

The item will be a weapon, an armor, a piece of jewelry or a book.

There is a 2% probability that the item is unique. If the item is not unique it will be magical.

Item Drop

There is a 26% probability that the item is gold and a 74% probability that it is any item, including gold.

There is a 2% probability that the item is unique.

Potion or Town Portal Drop

On dungeon level 1 there is a 50% probability that the item is a potion of healing and a 50% probability that it is a potion of mana.

On all other dungeon levels there is a 33% probability that the item is a potion of healing, a 33% probability that it is a potion of mana and a 33% probability that it is a scroll of town portal.

Trapped Chests

There is a 10% probability that a chest is rigged with a trap.

In the Catacombs the trap is one of:

  • Arrow
  • Fire Arrow

In all other dungeon types the trap is one of:

  • Arrow
  • Fire Arrow
  • Nova
